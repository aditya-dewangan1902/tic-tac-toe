# Tic Tac Toe – C Program

A simple 2-player Tic Tac Toe game in terminal.

## How to Run

```
gcc tic_tac_toe.c -o tic_tac_toe
./tic_tac_toe
```

## Gameplay

• 3×3 board  
• Players take turns: X then O  
• Enter position from 1-9  
• Game ends when a player wins or board is full
